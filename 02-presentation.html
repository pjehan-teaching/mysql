Présentation

  • Une base de données permet de stocker des informations de façon structurée.
  • Permet de rendre un site web dynamique.
  • On peut communiquer avec une base de données à l’aide du langage SQL.
  • La base de données se trouve sur un serveur et est interrogée par des clients.

Principaux SGBDR - MySQL

SGBDR le plus utilisée pour le développement web. Il est notamment mis en place sur des sites internet tels que Wikipedia, Facebook, Google ou Twitter et par les CMS les plus populaires (Wordpress, Drupal, Joomla…).

Principaux SGBDR - MariaDB

MariaDB est un “fork” de MySQL créé en 2009 suite au rachat de MySQL par Oracle. Les développeurs de ce projets sont ceux qui ont contribué à la conception de MySQL.

Principaux SGBDR - PostgreSQL

SGBDR open-source très performant et robuste. Utilisé par de nombreuses entreprises (Apple, Cisco, Fujitsu, Skype…).

Il est souvent préféré à MySQL pour des applications nécessitant de gérer un très grand volume de données ou des opérations complexes.

Principaux SGBDR - SQL Lite

SGBDR utilisé principalement sur les systèmes embarqués (smartphone, tablette...). SQLite n’utilise PAS le modèle client-serveur.

Principaux SGBDR - Oracle

Oracle est une base de données payante utilisée principalement par les entreprises. Oracle est très performant lorsque la base de données contient un très grand nombre de données.

Principaux SGBDR - SQL Server / Access

Microsoft propose un SGBDR payant SQL Server mais il existe également une version express gratuite. SQL Server est souvent utilisé sur les serveurs IIS pour les sites web développés en ASP.NET. Access est également un SGBDR rudimentaire mais facile à prendre en main.

MySQL (prononcer My Sequel en anglais)

  • 1ère version en 1995
  • Code source sous licence GNU (permet d’utiliser, d’étudier, de partager et de modifier l’application)
  • Version payante disponible (support technique et fonctionnalités supplémentaires).
  • Depuis le rachat par Oracle, ce SGBDR connaît un léger déclin et de nombreux concurrents viennent perturber sa supériorité.
