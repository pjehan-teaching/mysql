Manipuler les données

Insert

Vous pouvez insérer des données dans une table en utilisant une requête INSERT INTO :

INSERT INTO ma_table (id, nom, prenom) VALUES (1, 'Dupont', 'Jean');
INSERT INTO ma_table (id, nom, prenom) VALUES (1, 'Dupont', 'Jean'), (2, 'Dupont', 'Eric');
L'erreur "Duplicate entry for primary key" signifie que vous avez inséré 2 fois la même clé primaire

Update

Vous pouvez modifier des données en utilisant une requête UPDATE :

UPDATE ma_table SET nom = 'Dupont' WHERE id = 2
N'oubliez pas la clause WHERE sinon l'ensemble des données seront modifiées

Delete

Vous pouvez supprimer des données en utilisant une requête UPDATE :

DELETE FROM ma_table WHERE id = 2
N'oubliez pas la clause WHERE sinon l'ensemble des données seront supprimées
